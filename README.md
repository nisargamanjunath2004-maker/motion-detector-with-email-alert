# motion-detector-with-email-alert

The Motion Detector with Email Alert is an IoT-based security project used to detect movement and send automatic email notifications to the user. In this project, a PIR Sensor detects human motion and sends the signal to the ESP8266 NodeMCU. The controller processes the signal and, through Wi-Fi, triggers an email alert using IFTTT.
When motion is detected, the LED turns ON as an indication, and an email notification is automatically sent to the registered user. This project is mainly used for smart security and surveillance systems.
Components Used – Short Note
ESP8266 NodeMCU – Main controller with Wi-Fi connectivity.
PIR Sensor – Detects human movement.
LED – Indicates motion detection.
220-ohm resistor – Protects the LED from excess current.
Breadboard – Used for circuit connections.
Jumper wires – Connect components together.
IFTTT – Used to send automatic email alerts.
